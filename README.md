# Detection and Response Portfolio

## About This Portfolio

This repository documents my practical work in detection engineering, security operations, and incident response.

I work as an IT Support Engineer in a Microsoft focused environment and I am building deeper detection and response skills through practical homelab projects.

The portfolio focuses on SIEM deployment, endpoint monitoring, log ingestion, threat hunting, alert investigation, MITRE ATT&CK mapping, custom detection logic, incident documentation, and response playbooks.

## Current Lab Environment

The current lab runs on a dedicated Lenovo ThinkCentre M710q Mini using Proxmox VE.

The first completed build includes:
